# ExMark

[[list
line parser
block parser
body parser
]]

line parser: [file:String -> lines:[String]]
    根据每行开始的符号来确定行作用
block parser: [lines:[String] -> blocks:[String]]
    将不同行拼接形成块 主要是对 block_start block_end 之间的 paragraph 进行拼接
    每个block必须有一个属性
body parser: [block.body -> render]
    对行内格式进行解析
    注意嵌套

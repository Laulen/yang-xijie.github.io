# GitHub Pages

简单来说，GitHub Pages是可以轻松快速部署静态网站的方式，你可以在GitHub的每一个仓库创建一个对应的静态网站，存放说明或文档。通俗来说，就是白嫖GitHub的服务器建站。

- https://pages.github.com
